----------------------------------------------------------------------------------------------------
This is a demon of the paper:

" Perception Coordination Network: A Neuro Framework for Multi-Modal Concept Acquisition and Binding" 

----------------------------------------------------------------------------------------------------
Training

Run the PCN.m in the package and answer the questions shown in the command window during 
learning.

You can save your learning result in the folder "testresult" to test your result.
----------------------------------------------------------------------------------------------------
Testing

Load your learning result in the folder "testresult".

Run the PCNAcall.m in the folder "test\Acall" to use audition call vision concepts.

Run the PCNVcall.m in the folder "test\Vcall" to use vision call audition concepts.

----------------------------------------------------------------------------------------------------

The Matlab image processing package and voice tool box are used. Please refer to:
http://www.imageprocessingplace.com/root_files_V3/software/software.htm
